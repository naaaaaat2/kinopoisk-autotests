# Проект автоматизации тестирования Кинопоиска

## 📋 Описание проекта
Автоматизация UI и API тестов для веб-сервиса Кинопоиск (kinopoisk.ru) на основе 
[тест-плана и отчета по ручному тестированию](https://nat212.yonote.ru/share/bb8d0169-bf1e-492e-9fbb-0c7497dd5e88)

## 🏗️ Структура проекта
- `api/` - клиент для работы с API
- `tests/` - автоматические тесты
  - `test_api.py` - API тесты (8 тестов)
  - `test_ui.py` - UI тесты (10 тестов)
- `pages/` - Page Object модели
- `config/` - конфигурационные файлы
- `utils/` - вспомогательные утилиты

## 🐛 Найденные дефекты
- **BUG-001**: API возвращает 400 вместо 404 для несуществующего ID фильма
- Статус: подтвержден, автоматизирован (xfail)
